# tailwind-event-template

A template build using [Tailwind](https://tailwindcss.com/) for an Event Booking App.

## Getting started

1. Clone the repository:

   ```bash
   git clone <repo-url>
   ```

2. Install the dependencies:

   ```bash
   # Using npm
   npm install

   # Using Yarn
   yarn
   ```

3. Start the development server:

   ```bash
   # Using npm
   npm run serve

   # Using Yarn
   yarn run serve
   ```
